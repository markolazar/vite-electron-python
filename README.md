# vite-electron-python

An Electron application with React and TypeScript and Python

## Recommended IDE Setup

- [VSCode](https://code.visualstudio.com/) + [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) + [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)

## Dependencies

### Node.js and npm
- Node.js version 18 or higher
- npm (included with Node.js)

### Python
- Python 3.14 or higher

### uv Python package manager
- uv: A fast Python package installer and resolver. Install from [https://github.com/astral-sh/uv](https://github.com/astral-sh/uv)

## Project Setup

### Install

```bash
npm init
```

### Development

```bash
npm run dev
```

### Build

```bash
# For windows
$ npm run build:win

# For macos
```
